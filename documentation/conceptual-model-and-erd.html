Conceptual Model

profile

  • profile_id (primary key)
  • profile_about
  • profile_activation_token (for account verification)
  • profile_email
  • profile_image_url
  • profile_hash (for account password)
  • profile_name

follow

  • follower_profile_id (foreign key)
  • following_profile_id (foreign key)

thread

  • thread_id (primary key)
  • thread_profile_id (foreign key)
  • thread_reply_thread_id (foreign key)
  • thread_content
  • thread_datetime
  • thread_image_url

like

  • like_profile_id (foreign key)
  • like_thread_id (foreign key)
  • like_datetime

keyword

  • keyword_id (primary key)
  • keyword_name

tag

  • tag_keyword_id (foreign key)
  • tag_thread_id (foreign key)

Relations

  • One profile can make many threads (1 to many).
  • Many threads can be liked by many profiles (many to many).
  • Many profiles can follow many profiles (many to many).
  • Many threads can be tagged with many keywords (many to many).
  • One thread can have many thread replies (one to many).
